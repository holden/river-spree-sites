river-spree-sites
=================

Jekyll river-spree Blogpost Website

Home

This is the Jekyll Website project of river-spree, a Berlin based Spree Commerce user group.
Install Jekyll

gem install jekyll
Clone Repository

git clone git@github.com/river-spree/river-spree-sites
Build

    cd river-spree-sites
    jekyll build
